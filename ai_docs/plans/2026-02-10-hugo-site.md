# Plan: Westside Youth Rugby Static Website

**Date:** 2026-02-10
**Status:** in-progress

## Context

Static site for Westside Youth Rugby (Portland, OR) using Hugo + PaperMod theme, deployed to GitHub Pages. Zero cost, volunteer-maintainable.

## Phases

### Phase 0: Install Hugo via Nix

- [ ] Add `hugo` to `~/.dotfiles/nix/home/default.nix` (between `hidapitester` and `jq`)
- [ ] Run `hms` to apply
- [ ] Verify: `hugo version` shows extended edition

### Phase 1: Hugo Project Scaffolding

- [ ] `hugo new site . --format=yaml` in repo root
- [ ] `git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod`
- [ ] Configure `hugo.yaml` (baseURL, title, theme, PaperMod params, menus)
- [ ] Add `.gitignore` (public/, resources/, .hugo_build.lock)
- [ ] Verify: `hugo server` runs without errors
- [ ] Commit: "Init Hugo site with PaperMod theme"

### Phase 2: Initial Content

- [ ] `content/_index.md` — homepage with welcome message
- [ ] `content/about.md` — club info, mission, location
- [ ] `content/contact.md` — placeholder contact info
- [ ] `content/news/_index.md` — section listing
- [ ] `content/news/2026-02-10-welcome.md` — sample post
- [ ] Add menu entries (Home, About, News, Contact) to `hugo.yaml`
- [ ] Verify: all pages render, menu navigation works
- [ ] Commit: "Add initial site content"

### Phase 3: GitHub Actions Workflow

- [ ] `.github/workflows/hugo.yaml` — build + deploy on push to main
  - Hugo extended, submodules: recursive, hugo --minify, deploy-pages
- [ ] Commit: "Add GitHub Actions Hugo deploy workflow"
- [ ] Push to origin/main
- [ ] Enable Pages (source: GitHub Actions) in repo settings
- [ ] Verify: site loads at github.io URL

### Phase 4: Documentation

- [ ] `AGENTS.md` (symlinked as `CLAUDE.md`) — project instructions for AI agents
- [ ] README with content editing guide (how to add news posts via GitHub web UI)
- [ ] Document custom domain setup (future reference, in README or ai_docs)
- [ ] Document CMS setup (future reference, in ai_docs)
- [ ] Commit: "Add site documentation"

## Out of Scope

- Custom domain, CMS layer, contact forms, analytics, team roster, photo galleries

## Verification

1. `hugo --minify` builds without errors

4. All pages navigable: Home → About → News → Contact
